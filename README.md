## Scripts Description

### Script 1: System Identity Report
Displays system information like user, kernel version, date, and uptime.

### Script 2: Package Inspector
Checks if a package is installed and displays its version and description.

### Script 3: Disk and Permission Auditor
Checks directories and displays their permissions and size.

### Script 4: Log File Analyzer
Reads a log file and counts occurrences of the word "error".

### Script 5: Open Source Manifesto Generator
Takes user input and generates a manifesto saved in a text file.

## How to Run Scripts

1. Give permission:
   chmod +x script.sh

2. Run:
   ./script.sh

Example:
   ./script4.sh testlog.txt
